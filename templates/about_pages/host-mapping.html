Host Maps in PoPS

How does PoPS use host maps?

Each application of PoPS uses a customized host map that provides the locations of target hosts in each grid cell across the modeled landscape.

Currently, PoPS users are responsible for loading host maps to run the model. Below are some resources for downloading host maps:

  • Forest data:
    • LEMMA (30m resolution data for tree species on West Coast)
    • USDA (FIA Raster maps of live tree species basal area at 250m resolution for entire USA)
  • Cropland data:
    • Google Earth Engine
    • NASS (30m yearly map of most productive commodity in that cell)

Future host maps in PoPS

We are currently developing a host mapping module to automate the creation of host maps in PoPS.
